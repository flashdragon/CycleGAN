# Drawing a Cat Using GAN

# 연구 배경
- ### 1인가구가 늘어서 요즘 반려동물을 키우는 인구의 증가
- ### AI 서비스가 대세인 요즘 새로운 AI 서비스를 만들자

# 관련 연구
- ### GAN(Generate Adverial Network): 가짜 이미지를 만드는 모델
- ### Cycle GAN : image to image translation을 하는 모델

## 프로젝트 구조

---

```
Front-end : React
Back-end : Flask
database : mysql
Library : tensorflow
```
